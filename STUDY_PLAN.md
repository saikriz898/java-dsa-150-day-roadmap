# 📚 Java + DSA Complete Study Plan (150 Days)

This document provides a **structured roadmap to learn Java and Data Structures & Algorithms in 150 days**.

The plan focuses on:

* Java programming fundamentals
* Object Oriented Programming
* Data Structures
* Algorithmic thinking
* Consistent coding practice

Recommended daily study time:

```
2 hours per day
```

Estimated duration:

```
150 days ≈ 5 months
```

Follow this structure every day.

| ---------- | --------------------- |

Example schedule:

