# 🚀 Ultimate Java + DSA Study Roadmap (150 Days)

This roadmap provides a **structured plan to learn Java and Data Structures & Algorithms** with consistent practice.

The goal of this roadmap is to:

* Build strong **Java programming fundamentals**
* Develop **logical thinking**
* Master **Data Structures & Algorithms**
* Improve **problem solving skills**

Recommended study time:

```
2 hours per day
≈ 150 days (5 months)
```

---

# ⏰ Daily Study Structure

Use this schedule every day.

| Time       | Activity         |
| ---------- | ---------------- |
| 20 minutes | Learn concept    |
| 60 minutes | Coding practice  |
| 20 minutes | Debugging        |
| 20 minutes | Review and notes |

Example:

```
5:00 – 5:20 → Learn topic
5:20 – 6:20 → Solve coding problems
6:20 – 6:40 → Debug & review
```

Consistency is more important than intensity.
